Windows Installation

Here comes the instruction from windows

{# installing steps #}

Step 1: Download the Installer

Go to the Download Page and click on the Windows installer link to download the SignAI file.

Step 2: Run the Installer

Locate the downloaded installer file (usually in your "Downloads" folder) and double-click it to run the installation wizard.

Step 3: Follow the Installation Wizard

Follow the on-screen instructions provided by the installation wizard. You may need to agree to the terms and conditions and choose an installation location.

System requirements

  • Windows 10/11 (64-bit)
  • At least 8 GB RAM
  • 500 MB free space
  • Webcam for translation or uploading

macOS Installation

Here comes the instruction from macOS

{# installing steps #}

Step 1: Download the Installer

Go to the Download Page and click on the macOS download link to get the SignAI.dmg file.

Step 2: Open the DMG File

Locate the downloaded SignAI.dmg file in your Downloads folder and double-click it to open the installation window.

Step 3: Install SignAI

In the window that appears, drag the SignAI icon into your Applications folder to install the app.

Step 4: Launch SignAI

Open Launchpad or go to your Applications folder and double-click SignAI to start the app.

Step 5: Allow Permissions (if needed)

If you see a security message saying the app is from an unidentified developer, go to System Settings → Privacy & Security and click “Open Anyway”.

System Requirements

  • macOS 12 (Monterey) or later
  • At least 8 GB RAM
  • 500 MB of free disk space
  • Camera or webcam for live sign translation or uploads

Linux Installation

Here comes the instruction from Linux

{# installing steps #}

Step 1: Download the Package

Go to the Download Page and download the appropriate package for your distribution:

  • Debian/Ubuntu: SignAI.deb
  • Fedora/RHEL: SignAI.rpm
  • Other distributions: SignAI.AppImage

Step 2: Install the Package

For .deb files (Debian/Ubuntu):

Open a terminal and run: sudo dpkg -i SignAI.deb

If there are dependency issues, run: sudo apt-get install -f

For .rpm files (Fedora/RHEL):

Open a terminal and run: sudo rpm -i SignAI.rpm

For .AppImage files:

Make the file executable: chmod +x SignAI.AppImage

Then run it: ./SignAI.AppImage

Step 3: Launch SignAI

You can launch SignAI from your application menu or by running signai in the terminal.

Step 4: Grant Camera Permissions

When first launching SignAI, make sure to grant camera permissions when prompted. This is required for sign language translation.

System Requirements

  • Linux Kernel 5.0 or later
  • At least 8 GB RAM
  • 500 MB of free disk space
  • Webcam or camera for live sign translation
  • Graphics driver with OpenGL 3.0+ support

Android Installation

Here comes the instruction from Android

{# installing steps #}

Step 1: Download the APK

Go to the Download Page and click on the Android download link to get the SignAI.apk file.

Step 2: Enable Unknown Sources

Before installing, you need to allow installation from unknown sources:

  • Go to Settings → Security (or Settings → Apps on newer Android versions)
  • Enable "Install unknown apps" or "Unknown sources" for your browser or file manager

Note: You can disable this after installation for security.

Step 3: Install the APK

Open your Downloads folder or notification area, tap on the SignAI.apk file, and follow the on-screen instructions to install.

Step 4: Grant Permissions

When you first launch SignAI, grant the following permissions:

  • Camera: Required for sign language translation
  • Storage: To save and load videos
  • Microphone: Optional, for audio features

System Requirements

  • Android 8.0 (Oreo) or later
  • At least 4 GB RAM
  • 200 MB of free storage
  • Camera for live sign translation

iOS Installation

Here comes the instruction from iOS

{# installing steps #}

Step 1: Download from App Store (Recommended)

The easiest way to install SignAI on iOS is through the App Store. Search for "SignAI" and tap Get to install.

Step 2: Alternative - TestFlight (Beta)

If you want to test the latest beta version:

  • Install TestFlight from the App Store (if you don't have it)
  • Get the TestFlight invitation link from our Download Page
  • Open the link on your iOS device and tap Accept
  • Open TestFlight and install SignAI

Step 3: Launch SignAI

Find the SignAI icon on your home screen and tap it to launch the app.

Step 4: Grant Camera Permissions

When you first open SignAI, you'll be asked to grant camera access. Tap Allow to enable sign language translation features.

System Requirements

  • iOS 14.0 or later
  • iPhone 7 or newer (iPhone 8+ recommended)
  • 200 MB of free storage
  • Camera for live sign translation
